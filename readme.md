# socket-io-legacy

> Re exported legacy socket libs


## Install

```
$ npm install @gradeup/socket-io-legacy
```


## Usage

```js
const { io, emitter, redis }= require('@gradeup/socket-io-legacy');
// io => socket.io
// emitter => socket.io-emitter
// redis => socket.io-redis
```
